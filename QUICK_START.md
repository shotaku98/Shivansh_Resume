## üé® Quick Customizations

### Add Your Photo
1. Save your professional photo as `profile.jpg` in the `images` folder
2. Open `index_v2.html` in a text editor
3. Find line ~130 (the "image-placeholder" div)
4. Replace with:
```html
<img src="images/profile.jpg" alt="Shivansh Aggarwal">
```

### Change Colors
1. Open `css/style_v2.css`
2. Find the `:root` section at the top
3. Change these values:
```css
--primary-color: #0066FF;    /* Your main color */
--secondary-color: #00D4FF;  /* Your accent color */
```

### Update Contact Info
Edit the contact section in `index_v2.html` (around line 1138)

## üÜò Troubleshooting

### Images not showing?
- Make sure `images` folder is in the same directory
- Check image file names match exactly

### Colors look different?
- Clear your browser cache (Ctrl+Shift+R or Cmd+Shift+R)

### JavaScript not working?
- Check browser console (F12 ‚Üí Console tab)
- Ensure `js/main_v2.js` exists

### Mobile menu not opening?
- JavaScript might not be loaded
- Check file paths in HTML
